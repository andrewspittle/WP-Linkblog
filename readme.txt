=== WP Linkblog ===
Contributors: andrewspittle, andrewheiss
Tags: permalinks, linked list, link blog
Requires at least: 3.0.4
Tested up to: 3.4.1

Redirects permalinks of WordPress posts with a custom field to the destination URL.

== Installation ==

1. Download the plugin from Github
2. Upload it to the wp-contents/plugins directory of your website. 
3. Activate it, and add linkblog_url to any post you want to redirect
